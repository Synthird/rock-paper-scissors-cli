# Rock Paper Scissors CLI

A CLI Rock Paper Scissors game.

## Installation

1. You must have [python](https://python.org) installed
2. Download [game.py](game.py)
3. Run the following command:

```Python
python game.py
```

If the previous command didn't work, try

```Python
python3 game.py
```
